## About

This is a single-file static portfolio built in plain HTML, CSS, and JavaScript — no frameworks, no build step, no dependencies. Deploys anywhere in seconds.

- **Hero** — positioning, availability status, social links

- **Contact** — booking link and direct email

## Tech

Single `index.html` file. No npm, no build, no framework. CSS animations via keyframes. Scroll reveal via IntersectionObserver. Smooth scroll via requestAnimationFrame.

## Deployment

Hosted on Vercel. Any commit to `main` triggers an automatic redeploy.

To update: edit `index.html` directly in GitHub and commit. Done.
